# Приложение для оценки эмоциональной тональности текста

Это приложение помогает определить эмоциональную тональность текста: вставьте свой txt-файл с текстом на английском языке или загрузите файл с текстом из нашего репозитория.

Затем выберите лексикон: AFINN или NRC. Первый показывает, насколько отрицательная или положительная лексика в тексте, а второй разделяет слова по отдельным эмоциям.

После того, как вы выбрали лексикон AFINN, можете составить облако основных позитивных и негативных слов в отдельном чанке. Для этого просто выберите чанк и нажмите на кнопку «Выбрать чанк».

После того, как вы выбрали лексикон NRC, вы также можете выбрать отдельный чанк и посмотреть, как внутри него соотносятся друг с другом эмоции. Порядок действий будет таким же.
